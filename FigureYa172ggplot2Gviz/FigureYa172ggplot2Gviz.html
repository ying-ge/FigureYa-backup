bigwigbedgraph

Convert BigWig to BedGraph

ATAC-seqbigwigbedgraph The ATAC-seq data provided by the authors is a BigWig file, which needs to be converted to a plain text file called BedGraph for plotting.

bigwigbedgraphbigWigTobedgraph To convert the BigWig file to BedGraph, you need the tool BigWigToBedGraph.

# bigwigtobedgraph
# Install bigwigtobedgraph
conda install -c bioconda ucsc-bigwigtobedgraph

# bigwigbedgraph
#bigwig to bedgraph
bigWigToBedGraph -chrom=chr16 -start=16858001 -end=16868347 GSM2863128_N06_0.RPKM_nor.bw 0_ATAC_Igll1.bedgraph
bigWigToBedGraph -chrom=chr16 -start=16858001 -end=16868347 GSM2863129_NO6_24.RPKM_nor.bw 24_ATAC_Igll1.bedgraph
bigWigToBedGraph -chrom=chr16 -start=16858001 -end=16868347 GSM2863130_NO6_72.RPKM_nor.bw 72_ATAC_Igll1.bedgraph
bigWigToBedGraph -chrom=chr16 -start=16858001 -end=16868347 GSM2863131_NO6_cd19pos.RPKM_nor.bw proB_ATAC_Igll1.bedgraph

Extract the track information for the desired location

Igll1Igll1.bed This example shows the region near the Igll1 gene. The gene location information is written to the Igll1.bed file, and the signal in this region is extracted for plotting.

bedtools intersect -wa -a GSM2863140_Input_NO8_24_mm9.bedGraph -b Igll1.bed> 0_EBF1_Igll1.bedgraph
bedtools intersect -wa -a GSM2863147_EBF1_NO14_24_mm9.bedGraph -b Igll1.bed> 24_EBF1_Igll1.bedgraph
bedtools intersect -wa -a GSM2863148_EBF1_NO14_72_mm9.bedGraph -b Igll1.bed> 72_EBF1_Igll1.bedgraph
bedtools intersect -wa -a GSM2863149_EBF1_NO14_cd19pos_mm9.bedGraph -b Igll1.bed> proB_EBF1_Igll1.bedgraph

Prepare the gene structure file

gtfexon The specific locations of gene exons can be found in the gtf file.

gencodemm9gtfhttps://www.gencodegenes.org/mouse/release_M9.htmlhttps://www.gencodegenes.org/human/ensemblhttp://asia.ensembl.org/info/data/ftp/index.html Mouse and human gene annotation files are typically generated using Gencode. The mm9 gtf file is available at: https://www.gencodegenes.org/mouse/release_M9.html. Human annotations are available at https://www.gencodegenes.org/human/. Other species can be downloaded from Ensembl: http://asia.ensembl.org/info/data/ftp/index.html.

Igll1exonIDENSMUST00000124649.1chr16:16860674-16863985 The Igll1 gene used in the example contains two exon transcripts, which we will also use. Its ID is ENSMUST00000124649.1 and its location is chr16:16860674-16863985. Next, we will extract the exons of this gene.

grep Igll1 gencode.vM9.annotation.gtf | grep ENSMUST00000124649.1 |grep exon | awk '{print $4,$5,$3}' OFS="\t" > Igll1.gene.txt
