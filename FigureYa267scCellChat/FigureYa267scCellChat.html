Comments from the author of this document, Xiao Gu:

Given the enrichment of macrophage subpopulations in liver metastasis, we hypothesize that those liver metastases enriched macrophages may be functionally distinct across How to test this hypothesis for different disorders?

The author uses CellPhoneDB analysis and circlize visualization to solve the problem. Due to CellPhoneDBs inability to directly handle the relationship between two groups, the author used an alternative approach:

-Combining all cell types of rectal cancer together, equivalent to one cell type, named tumor, can overcome the difficulty of cellphonedb in comparing the relationship between two groups at once.

I actually dont quite agree with this method, as it erases information about the cell types of colon cancer. I personally believe that Cellchats algorithm is superior to Cellphonedb, especially in analyses with control processing.

The friend who raised the request requested to use Cellchat to implement it, which can indeed be very convenient. See Interpretation of Results at the end of Start Drawing for details.

Input data preprocessing

ifnb - A Seurat object with the PBMC control/IFNB-stimulated dataset From< https://pubmed.ncbi.nlm.nih.gov/29227470/ >, has been packaged into SeuratData, we can directly install and load it.

The example data ifnb has two groups: STIM and CTRL:

-First, split it into two seurat objects, normalize the two datasets separately and identify their respective highly variable genes. Use the highly variable genes to perform PCA on the two gene sets separately. -Then integrate the two datasets, perform regular data preprocessing, and store them in immunembined for subsequent analysis.

##  (Load example dataset)
# ifnbIFN-PBMCRNA-seq
# Install and load the 'ifnb' dataset containing single-cell RNA-seq data of IFN- stimulated PBMCs
# InstallData("ifnb")

data("ifnb")
ifnb <- UpdateSeuratObject(ifnb)
?ifnb  #  (View dataset documentation)

# ifnbCTRLSTIM
# The 'ifnb' dataset contains two conditions: CTRL (control) and STIM (stimulated)
head(ifnb@meta.data)  #  (View first few rows of metadata)
table(ifnb@meta.data$stim)  #  (Count cells in each condition)


## (STIMCTRL)
# splitobjectifnbseurat(STIMCTRL)
# Split the dataset into two Seurat objects based on stimulation condition (STIM vs CTRL)
ifnb.list <- SplitObject(ifnb, split.by = "stim")

# normalize
# Normalize each dataset and identify highly variable features (genes)
ifnb.list <- lapply(X = ifnb.list, FUN = function(x) {
  x <- NormalizeData(x)  #  (Normalize gene expression values)
  x <- FindVariableFeatures(x, selection.method = "vst", nfeatures = 2000)  #  (Identify highly variable genes)
})

# ranktop2000
# Select features for integration (top 2000 variable genes across both datasets)
features <- SelectIntegrationFeatures(object.list = ifnb.list, nfeatures = 2000)

# PCA
# Perform PCA on each dataset using the selected features
ifnb.list <- lapply(X = ifnb.list, FUN = function(x) {
  x <- ScaleData(x, features = features, verbose = T)  #  (Scale data)
  x <- RunPCA(x, features = features, verbose = T)  #  (Perform PCA)
})


## 
# findinintegrationanchorsIntegrateData
# k.anchor5
# Identify integration anchors and integrate the two datasets
# Adjust k.anchor to control integration strength (higher values = stronger integration)
immune.anchors <- FindIntegrationAnchors(object.list = ifnb.list, anchor.features = features, 
                                         reduction = "rpca", k.anchor = 20)
immune.combined <- IntegrateData(anchorset = immune.anchors)  #  (Integrate datasets)

# 
# Set the default assay to the integrated data for downstream analysis
DefaultAssay(immune.combined) <- "integrated"

# scalePCA30PCUMAPNeighborscluster
# Standard workflow: scale data, run PCA, UMAP, neighbor graph construction, and clustering
immune.combined <- ScaleData(immune.combined, verbose = FALSE)  #  (Scale integrated data)
immune.combined <- RunPCA(immune.combined, npcs = 30, verbose = FALSE)  # PCA (PCA dimensionality reduction)
immune.combined <- RunUMAP(immune.combined, reduction = "pca", dims = 1:30)  # UMAP (UMAP visualization)
immune.combined <- FindNeighbors(immune.combined, reduction = "pca", dims = 1:30)  #  (Construct k-nearest neighbor graph)
immune.combined <- FindClusters(immune.combined, resolution = 0.5)  #  (Cluster cells)

# RNA assay

## STIM (STIM Group)
# STIM (Subset cells from the stimulated (STIM) group)
stim.object <- subset(immune.combined, stim == "STIM")
# RNACellChat (Retrieve normalized RNA expression matrix for CellChat)
stim.data.input <- GetAssayData(stim.object, assay = "RNA", slot = "data")
#  (Extract cell type annotations and stimulation condition)
stim.meta <- stim.object@meta.data[, c("seurat_annotations", "stim")] 
# CellType (Ensure CellType is stored as a vector)
stim.meta$CellType %<>% as.vector(.)
# CellChat (Initialize CellChat object for communication analysis)
stim.cellchat <- createCellChat(object = stim.data.input)
# CellChat (Add cell metadata to CellChat object)
stim.cellchat <- addMeta(stim.cellchat, meta = stim.meta)
# Seurat (Set cell identities to Seurat annotations)
stim.cellchat <- setIdent(stim.cellchat, ident.use = "seurat_annotations")

# CellChat (Use human ligand-receptor database provided by CellChat)
stim.cellchat@DB <- CellChatDB.human 
#  (Subset database based on available cell types)
stim.cellchat <- subsetData(stim.cellchat) 
# 10 (Set parallel computation with 10 workers)
future::plan("multisession", workers = 10) 
#  (Identify overexpressed genes in each cell type)
stim.cellchat <- identifyOverExpressedGenes(stim.cellchat)
#  (Identify overexpressed ligand-receptor pairs)
stim.cellchat <- identifyOverExpressedInteractions(stim.cellchat)
#  (Project data onto human PPI network)
stim.cellchat <- smoothData(stim.cellchat, adj = PPI.human)  

#  (Compute communication probabilities between cells)
stim.cellchat <- computeCommunProb(stim.cellchat)
#  (Optional: Filter out low-confidence interactions)
# stim.cellchat <- filterCommunication(stim.cellchat, min.cells = 10)

#  (Compute pathway-specific communication probabilities)
stim.cellchat <- computeCommunProbPathway(stim.cellchat)
#  (Aggregate communication probabilities into a network)
