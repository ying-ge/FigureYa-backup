输入文件
Input file
三个输入文件，其中第一个文件必需提供，第二和第三个文件非必需：
karyotype.csv，用于画染色体轮廓。包含五列：第一列为染色体名，第二、三列为染色体的起始和终止点。如果能提供着丝粒的开始和结束位置，就给出第四列（起始）和第五列（终止），如果着丝粒位置未知，就只给前三列。0-based基因组坐标(以0-based开始、1-based结束)。
easy_input_heatmap.csv，用来绘制染色体heatmap。包含四列，第一列染色体名，第二、三列为染色体上的位置，第四列为相应位置上的值。例如差异基因的log2foldchange、甲基化水平等。
easy_input_mark.csv，如果想画染色体旁边的形状标记，就要提供这个文件。包含六列，第一列type，第二列形状（只支持三种形状：box正方形、triangle三角形和circle圆。如果需要标记的特征超过3种，就用第六列里的颜色区分吧，例如红色的三角形，蓝色的三角形等），第三列染色体，第四、五列染色体上的位置，第六列颜色。
注意：
同一type对应的颜色一致。
Three input files, of which the first is required and the second and third are not required:
karyotype.csv, used for drawing chromosome outlines. It contains five columns: the first column is the chromosome name, and the second and third columns are the start and end points of the chromosome. If the start and end positions of the centromere are available, the fourth column (start) and the fifth column (end) are included; if the centromere position is unknown, only the first three columns are provided. 0-based genomic coordinates (start with 0-based and end with 1-based).
easy_input_heatmap.csv, used for drawing chromosome heatmaps. It contains four columns: the first column is the chromosome name, the second and third columns are the positions on the chromosome, and the fourth column is the value at the corresponding position, such as the log2 fold change of differential genes, methylation levels, etc.
easy_input_mark.csv: If you want to draw shape markers next to the chromosomes, you need to provide this file. It contains six columns: the first column is the type, the second column is the shape (only three shapes are supported: box, triangle, and circle. If the features you need to mark exceed three types, use the color in the sixth column to differentiate, such as red triangles, blue triangles, etc.), the third column is the chromosome, the fourth and fifth columns are the positions on the chromosome, and the sixth column is the color.
Note:
The color corresponding to the same type should be consistent.

开始画图
Start drawing
ideogram(karyotype = karyotype, 
         
         #如果没提供easy_input_heatmap.csv，就在下面两行前面加#
         #If easy_input_heatmap.csv is not provided, precede the following two lines with#
         overlaid = gene_density, 
         #染色体上heatmap的配色
         #color matching of heatmap on chromosomes
         colorset1 = c("forestgreen", "gold", "deeppink3"), 
         
         #如果没提供easy_input_mark.csv，就在下面这行前面加#
         #If easy_input_mark.csv is not provided, prefix the following line with#
         label = Random_RNAs_500, 
         label_type = "marker",
         
         #染色体宽度，默认170
         #chromosome width, default 170
         width = 180, #注意：太宽会跑出画布外面 Note: If it's too wide, it will extend beyond the canvas
         
         #图例的位置
         #position of legend
         Lx = 160, #图例左上角跟左边的距离 distance from the top left corner of the legend to the left side
         Ly = 20, #图例左上角跟顶端的距离 distance from the upper left corner of the legend to the top 
         
         #输出文件
         #output file
         output = "chromosome.svg") #只支持svg格式 only svg format is supported

# 可以转成png，方便查看
# It can be converted to png for easy viewing
convertSVG("chromosome.svg", device = "png")

附：karyotype.csv文件的准备
Appendix: Preparation of karyotype.csv file
这里提供两种两种方法，二选一即可。  Two methods are provided here, choose one.
方法一：用rtracklayer实现
Method 1: Using rtracklayer
适合处理复杂的基因组数据  Suitable for processing complex genomic data
library(rtracklayer)
library(dplyr)

# 读取 GTF 文件 \ Read GTF file
gtf <- import("your_file.gtf")  # 使用 rtracklayer 导入 GTF 文件 \ Use rtracklayer to import GTF file

# 提取染色体信息 \ Extract chromosome information
karyotype <- gtf %>%
  as.data.frame() %>%  # 转换为数据框 \ Convert to data frame
  filter(type == "chromosome") %>%  # 筛选染色体信息 \ Filter chromosome information
  select(seqnames, start, end) %>%  # 提取染色体名称、起始位置和终止位置 \ Extract chromosome name, start position, and end position
  rename(Chromosome = seqnames, Start = start, End = end) %>%
  mutate(Centromere_Start = NA, Centromere_End = NA)  # 如果没有着丝粒信息，可以填充 NA \ If centromere information is unavailable, fill with NA

# 如果有着丝粒信息，提取并合并 \ If centromere information is available, extract and merge
centromere <- gtf %>%
  as.data.frame() %>%
  filter(type == "centromere") %>%  # 筛选着丝粒信息 \ Filter centromere information
  select(seqnames, start, end) %>%
  rename(Chromosome = seqnames, Centromere_Start = start, Centromere_End = end)

karyotype <- karyotype %>%
  left_join(centromere, by = "Chromosome")

# 保存为 karyotype.csv \ Save as karyotype.csv
write.csv(karyotype, "karyotype.csv", row.names = FALSE)
方法二：用dplyr实现
Method 2: Using dplyr
适合结构简单的GTF文件  Suitable for simple GTF file structures
library(dplyr)

# 读取 GTF 文件 \ Read GTF file
gtf <- read.table("your_file.gtf", sep = "\t", header = FALSE, stringsAsFactors = FALSE)

# 提取染色体信息 \ Extract chromosome information
karyotype <- gtf %>%
  filter(V3 == "chromosome") %>%  # 筛选染色体信息 \ Filter chromosome information
  select(V1, V4, V5) %>%         # 提取染色体名称、起始位置和终止位置 \ Extract chromosome name, start position, and end position
  rename(Chromosome = V1, Start = V4, End = V5) %>%
  mutate(Centromere_Start = NA, Centromere_End = NA)  # 如果没有着丝粒信息，可以填充 NA \ If centromere information is unavailable, fill with NA

# 如果有着丝粒信息，提取并合并 \ If centromere information is available, extract and merge
centromere <- gtf %>%
  filter(V3 == "centromere") %>%  # 筛选着丝粒信息 \ Filter centromere information
  select(V1, V4, V5) %>%
  rename(Chromosome = V1, Centromere_Start = V4, Centromere_End = V5)

karyotype <- karyotype %>%
  left_join(centromere, by = "Chromosome")

# 保存为 karyotype.csv \ Save as karyotype.csv
write.csv(karyotype, "karyotype.csv", row.names = FALSE)
