### 1. Preparation

### 2. Make Changes
- Follow existing code style
- Add or modify files in appropriate directories
- Test your changes

### 3. Commit Code
```bash

```

### 4. Create Pull Request
- Use clear title
- Describe your changes in detail
- Link relevant Issues
